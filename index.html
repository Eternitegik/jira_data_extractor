Описание:

Расширение позволяет быстро скопировать нужные поля со страницы Jira.
Поддрерживаются следующие страницы:
 - jira.dear.com.ru
 - jira.rtk-sr.tech

 Выбрать поля для копирования и их порядок можно при нажатии на значек расширения в списке 
 расширений.

 Спиок доступных полей:
 Заголовок, Ссылка, Исполнитель, Автор, Статус, Тип задачи

Установка расширения

1) Скачайте последнюю версию расширения
2) Распакуйте архив в удобное место
3) Откройте браузер Google Chrome
4) Перейдите на страницу управления расширениями — chrome://extensions/
5) Сверху установите флажок "Режим разработчика"
6) Нажмите на появившуюся кнопку "Загрузить распакованное расширение"
7) Выберите папку со скачанным расширением

Обновление расширения

1) Скачайте последнюю версию расширения
2) Откройте папку, где хранится старое расширение
3) Удалите файлы из папки и перенесите в неё файлы из архива
