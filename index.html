Описание:

Расширение позволяет в два нажатия скопировать нужные поля со страницы Jira.
По умолчанию поддерживаются следующие страницы:
- jira.dear.com.ru
- jira.rtk-sr.tech

Выбрать поля для копирования и их порядок можно при нажатии на иконку расширения в списке 
расширений.

Список доступных полей:
Заголовок, Ссылка, Исполнитель, Автор, Статус, Тип задачи.

Если вам необходимо, что бы расширение работало и на других доменах jira, то вам необходимо скорректировать список доменов в файле manifest.json: content_scripts -> matches

Расширение только для google chrome.

Установка расширения

1) Скачайте последнюю версию расширения.
2) Создайте новую пустую папку в удобном для вас месте.
3) Из скаченного архива переместите содержимое в созданную ранее папку.
4) Откройте браузер Google Chrome.
5) Скопируйте — chrome://extensions/, откройте в новой вкладке браузера.
6) Сверху справа установите флажок "Режим разработчика". В левой части появится кнопка "Загрузить распакованное расширение".
7) Нажмите на кнопку "Загрузить распакованное расширение".
8) Выберите папку, созданную на шаге 2.

Обновление расширения

1) Скачайте последнюю версию расширения.
2) Откройте папку, где хранится старое расширение.
3) Удалите файлы из папки и перенесите в неё файлы из архива.
4) Скопируйте — chrome://extensions/, откройте в новой вкладке браузера.
5) У расширения нажмите на кнопку Обновить.
6) Нажмите на иконку в списке расширений и настройте.
